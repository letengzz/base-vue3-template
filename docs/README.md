# 快速开始

## 安装依赖

```bash
pnpm install
```

## 开发模式

```bash
pnpm dev
```

## 构建生产版本

```bash
pnpm build
```

## 类型检查

```bash
pnpm type-check
```

## 代码检查与修复

```bash
pnpm lint
```

## 代码格式化

```bash
pnpm format
```

## 运行测试

```bash
# 单元测试
pnpm test:unit

# E2E 测试
pnpm test:e2e
```
